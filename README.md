## :clapper: Usage
  Edit install.sh/setup.sh to customize

  To install & setup everything:
```bash
  git clone https://github.com/andreystarkov/humanization
  cd humanization
  bash ./install.sh
```

  Setup everything:
```bash
  bash ./setup.sh
```
